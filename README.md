# 静康享宁家政服务 — 官方网站

## 使用前需要修改的信息

打开 `index.html`，搜索下面的标记，替换成你的真实信息：

| 替换内容 | 搜索关键词 | 说明 |
|---------|-----------|------|
| 📞 客服电话 | `请填写您的联系电话` | 改成你的手机号 |
| 🏛️ 统一社会信用代码 | `请填写您的统一社会信用代码` | 营业执照上的号 |

## 如何查看效果

直接双击 `index.html` 在浏览器打开即可。

## 如何发布上线

### 方式一（最简单）：托管到 Vercel（免费）

```bash
npm install -g vercel
cd ~/.openclaw/workspace/jingkang
vercel --prod
```

### 方式二：托管到 GitHub Pages

1. 把 `jingkang` 文件夹上传到 GitHub 仓库
2. 仓库 Settings → Pages → 选择 main 分支
3. 几分钟后就能通过 `https://你的用户名.github.io/仓库名` 访问

### 方式三：用你自己的域名 + 服务器

把整个 `jingkang` 文件夹上传到服务器的 web 目录即可。
